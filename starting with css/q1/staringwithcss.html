CSS

CSS can be added in 3 types

  1. external css
  2. internal css
  3. inline css
if we arrange them in priority order then the order goes like this
inline>internal>external

Inline CSS

Inline CSS is a way of applying CSS styling directly within HTML elements using the style attribute. For example, you can change the color and text alignment of a heading by adding the style attribute to the < h1> tag, like this:
<h1 style=“color:blue;text-align:center;”>This is a heading</h1 >
 Inline CSS is useful for quick and simple styling changes, but it has some disadvantages. For instance, inline CSS cannot be reused for other elements, and it cannot style pseudo-elements or pseudo-classes. Inline CSS also mixes presentation with content, which makes the HTML code less readable and maintainable. Therefore, inline CSS is not recommended for large or complex web pages.

Internal CSS

Internal CSS is a method of adding CSS rules directly to the head section of an HTML document. It involves using the <style> element to enclose the CSS code within the HTML document itself12. Internal CSS is used to provide a unique style for a single HTML document23. For example, you can change the background color and font size of the body element by adding the style attribute to the <style> element, like this:
<style> /* Internal CSS */ body { background-color: lightblue; font-size: 20px; }

External

External CSS is a way of applying CSS styling to multiple HTML pages using a single style sheet file. External CSS has a .css extension and contains CSS rules for different HTML elements. External CSS can be linked to an HTML document using the <link> element inside the <head> section
